# Notes App 📝

A React.js based Notes App where users can create, manage, and delete notes easily.

## 🚀 Features
- Add multiple notes
- Delete notes anytime
- Simple and clean UI

## 🛠 Tech Stack
- React.js
- JavaScript
- CSS
- Vite

## ▶️ How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run: npm install
4. Run: npm run dev
